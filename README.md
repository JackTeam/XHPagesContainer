XHPagesContainer
================

XHPagesContainer 是一款仿真网易新闻App的交互框架.              
有以下功能：      
1、侧滑组件.             
2、套在scrollView里面的重用contentView组件.        
3、管理栏目组件.       
4、栏目选择组件.          
5、手势滑动返回上一层、下一层组件.
